## A TaskTracker Program
This is a project that follows these [requirements](https://roadmap.sh/projects/task-tracker).
It was constructed with a module design.  
To execute this **program** run this command:  

`java -jar taskTracker <options>`
        
Where **options** are:  
  - `add <title>`  
  - `delete <id>`  
  - `update <id> <title>`  
  - `mark-done <id>`  
  - `mark-in-progress <id>`  
  - `list`
  - `list to-do`
  - `list done`
